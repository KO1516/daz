📘 How to Understand This Dashboard

  • User Rating Distribution: This shows how many games got each rating score. A taller bar means more games got that rating.
  • Top 10 Genres by Count: "Count" means how many games are in that genre. Action might have 150 games, for example.
  • Average Rating by Platform: See which gaming platforms have higher-rated games on average.
  • Top Genres by Average Rating: See which genres players enjoy the most, based on their average ratings.

This dashboard was made to be fun and simple — so even a 10-year-old can explore and learn! 🎮
